1. INTRODUCCIÓN

Nombre de la aplicación
La aplicación se denomina, de forma genérica en este documento, como "Sistema de Punto de Venta e Inventario para Empresa". En la propia aplicación puede mostrarse el nombre comercial de tu empresa en la barra superior.

Qué hace la aplicación
La aplicación permite gestionar las operaciones básicas de un negocio: control de inventario de productos, registro de compras (adquisiciones), registro de ventas, manejo de caja, gestión de devoluciones, administración de proveedores, categorías de productos, usuarios, estadísticas e historial de ventas, así como la configuración de parámetros fiscales (SAR).

Para qué sirve
Sirve para llevar un control ordenado y centralizado de:
- Los productos que vende la empresa.
- Las existencias (stock) en inventario.
- Las compras a proveedores.
- Las ventas realizadas a clientes y los comprobantes de venta.
- Los movimientos de caja.
- Las devoluciones de productos.
- Los usuarios autorizados y sus permisos.
- Las estadísticas de las operaciones.

A quién está dirigida
Está dirigida a personal administrativo y operativo de la empresa, por ejemplo:
- Cajeros(as).
- Encargados(as) de inventario.
- Encargados(as) de compras.
- Supervisores y administradores.
No se requiere conocimiento de programación; sólo conocimientos básicos de uso de computadora y del funcionamiento del negocio.

2. REQUISITOS DEL SISTEMA

Sistema operativo compatible
- Windows 10 o superior (recomendado Windows 10, 11 o posterior).

Requisitos mínimos recomendados
- Procesador: Intel i3 o equivalente.
- Memoria RAM: 4 GB como mínimo (8 GB recomendado).
- Espacio en disco: al menos 500 MB libres para la aplicación y la base de datos.
- Resolución de pantalla: 1366 x 768 o superior (la aplicación está optimizada también para resoluciones mayores, como 1920 x 1080).
- Permisos para instalar y ejecutar aplicaciones en Windows.

-------------------------------------------------------------

3. INSTALACIÓN Y EJECUCIÓN

Según cómo se haya entregado la aplicación en tu empresa, podrás encontrarla de una de estas formas:
- Como ejecutable instalado en Windows (acceso directo en el Escritorio o en el menú Inicio).
- Como carpeta de aplicación portátil (no requiere instalación, sólo ejecutar el archivo principal).

Pasos generales para ejecutar la aplicación en Windows:
1. Localiza el acceso directo de la aplicación en el Escritorio o en el menú Inicio. Si la aplicación es portátil, busca la carpeta que te proporcionó el área de sistemas y dentro de ella el archivo ejecutable (por ejemplo, "Empresa.exe" o similar).
2. Haz doble clic sobre el icono del programa.
3. Espera unos segundos mientras se abre la ventana principal. Por defecto, la aplicación se abre en modo de pantalla grande (maximizada) para un mejor aprovechamiento del espacio.
4. Una vez abierta, aparecerá la pantalla de inicio de sesión.

Si la aplicación no abre:
- Verifica que tu usuario de Windows tenga permisos para ejecutar aplicaciones.
- Verifica con el área de sistemas que el archivo no haya sido bloqueado por el antivirus.

A continuación se describe cada pantalla principal de la aplicación, qué muestra y qué puede hacer el usuario en cada una.

4.1 Pantalla de inicio de sesión (Login)

Nombre de la vista o pantalla
- Inicio de sesión.

Descripción de su función
- Permite que los usuarios registrados ingresen al sistema mediante su correo y contraseña.
- En el primer uso, si no existen usuarios, guía la creación del usuario administrador inicial.

Qué información muestra
- Título "Inicio de sesión".
- Campos de:
  - Correo electrónico.
  - Contraseña.
- Botón para iniciar sesión.
- Enlace para recuperar o restablecer contraseña.

Qué botones tiene y qué hace cada uno
- Botón "Iniciar sesión": valida las credenciales; si son correctas y el usuario está activo, entra al menú principal.
- Icono de "ojo" en el campo de contraseña: muestra u oculta la contraseña para verificar lo tecleado.
- Enlace "¿Olvidaste tu contraseña?": abre la pantalla de restablecimiento de contraseña.
- (Sólo la primera vez, si no hay usuarios) Navega a una pantalla para crear el primer usuario administrador.

Qué puede hacer el usuario en esta pantalla
- Escribir su correo electrónico y contraseña.
- Cambiar la visibilidad de la contraseña.
- Iniciar sesión.
- Ir al proceso de recuperación de contraseña en caso de olvido.

4.4 Pantalla Inventario

Nombre de la vista o pantalla
- Inventario.

Descripción de su función
- Permite consultar, buscar, filtrar y administrar los productos registrados en el sistema.
- Permite agregar nuevos productos y generar reportes en PDF del inventario.

Qué información muestra
- Título "Inventario".
- Barra de búsqueda de productos por nombre o por código.
- Lista o cuadrícula de productos, mostrando por cada uno:
  - Nombre del producto.
  - Unidad de medida.
  - Precio de venta.
  - Costo.
  - Stock (cantidad disponible).
  - Estado (activo/inactivo, según se muestre).
- Opciones adicionales para:
  - Agregar producto.
  - Ver valor total del inventario.
  - Exportar el inventario a PDF.

Qué botones tiene y qué hace cada uno
- Campo de búsqueda con icono de lupa:
  - Al escribir, filtra los productos por nombre o código.

  - "Agregar producto": abre el formulario de nuevo producto.

  - "Exportar inventario PDF": genera un documento PDF con el listado de productos.

  - Botón con icono "+" para agregar el primer producto.
- Al hacer clic en un producto:
  - Abre la pantalla de perfil/detalle del producto, donde se puede ver más información y realizar acciones (editar, etc.).

Qué puede hacer el usuario en esta pantalla
- Buscar productos por nombre o código.
- Escanear códigos de barras (en Android/iOS).
- Revisar el stock y precios de los productos.
- Ver el valor total del inventario.
- Exportar el inventario a un PDF.
- Abrir el detalle de un producto para ver más información o editarlo (según permisos).

4.7 Pantalla Ventas

Nombre de la vista o pantalla
- Ventas.

Descripción de su función
- Permite registrar ventas a clientes.
- Permite seleccionar productos, cantidades, aplicar descuentos y generar la factura o comprobante.

Qué información muestra
- Título "Ventas".
- Si no hay una caja abierta, muestra una pantalla de "Caja cerrada" que indica que es necesario abrir una caja antes de vender.
- Cuando hay caja abierta, muestra:
  - Lista de productos seleccionados para la venta (carrito).
  - Para cada producto:
    - Nombre.
    - Unidad de medida.
    - ISV.
    - Precio, descuento aplicado (si lo hay) y subtotal.
    - Cantidad seleccionada.
  - Resumen de venta:
    - Total de productos.
    - Subtotal.
    - Descuento total.
    - Total a pagar.
  - Campos para completar la venta:
    - Método de pago (efectivo, tarjeta, transferencia).
    - Total.
    - Nombre del cliente (opcional).
    - RTN del cliente (opcional u obligatorio según el monto).
    - Monto recibido.
    - Cambio.

Qué botones tiene y qué hace cada uno
- Botón flotante (en móviles) para escanear código de barras y agregar productos:
  - Usa la cámara para escanear el código de productos y agregarlos a la venta.
- Buscador de productos (en escritorio):
  - Permite filtrar productos por nombre para agregarlos al carrito.
- Botón "+" y "-" en cada producto del carrito:
  - Aumentan o disminuyen la cantidad del producto en la venta.
- Botón de descuento (icono de etiqueta) en cada producto:
  - Abre un cuadro para aplicar un descuento porcentual al producto.
- Botón de papelera en cada producto:
  - Permite eliminar el producto del carrito, previa confirmación.
- Botón "Finalizar venta":
  - Abre un formulario de confirmación donde se llenan los datos de pago.
  - Valida que haya al menos un producto.
- En el formulario de finalizar venta:
  - Botón "Cancelar": cierra el formulario sin registrar la venta.
  - Botón "Confirmar": registra la venta, genera el número de factura y actualiza el inventario y la caja.
- Después de registrar la venta:
  - Se muestra un mensaje de éxito.
  - Puede mostrarse un cuadro preguntando si se desea generar la factura para impresión.

Qué puede hacer el usuario en esta pantalla
- Seleccionar productos para vender.
- Ajustar cantidades y aplicar descuentos.
- Ver el resumen del total a pagar.
- Ingresar datos del cliente (cuando sea necesario).
- Elegir el método de pago.
- Registrar la venta y, si corresponde, imprimir o generar la factura.

5. GUÍA DE USO PASO A PASO

En esta sección se explica cómo utilizar la aplicación desde que se abre hasta completar las funciones principales.

5.1 Iniciar la aplicación y acceder al sistema
1. Abre la aplicación desde el acceso directo o ejecutable en Windows.
2. Espera a que aparezca la pantalla de inicio de sesión.
3. Ingresa tu correo electrónico y contraseña.
4. Haz clic en "Iniciar sesión".
5. Si los datos son correctos, accederás al menú principal.
6. Si olvidas tu contraseña, haz clic en "¿Olvidaste tu contraseña?" y sigue las instrucciones en la pantalla de restablecimiento.

5.2 Navegar por el menú principal
1. Una vez dentro, verás el panel principal con tarjetas de acceso rápido a los módulos.
2. Abre el menú lateral (si está cerrado) para ver todas las opciones disponibles según tus permisos.
3. Haz clic en la sección que desees usar: por ejemplo, "Inventario", "Ventas", "Caja", etc.
4. Si lo deseas, activa el "Modo noche" para cambiar la apariencia a tema oscuro.

5.3 Gestionar inventario (alta y consulta de productos)

Consultar y buscar productos:
1. Desde el menú principal, selecciona "Inventario".
2. En la parte superior, usa la barra de búsqueda para escribir el nombre o el código del producto que deseas encontrar.
3. En dispositivos móviles compatibles, también puedes usar el icono de código de barras para escanear el producto.
4. Revisa la lista de resultados, donde verás nombre, unidad de medida, stock, precio y costo de cada producto.
5. Haz clic en un producto para ver su detalle completo.

Agregar un nuevo producto:
1. En la pantalla de Inventario, abre el menú de opciones (icono de tres puntos) o, si no hay productos, usa el botón "+".
2. Selecciona "Agregar producto".
3. Se abrirá el formulario de producto.
4. Completa los campos obligatorios (código, nombre, unidad, precio, costo, proveedor, categoría, etc.).
5. Revisa que los datos sean correctos.
6. Haz clic en "Guardar".
7. El nuevo producto aparecerá en la lista de inventario.

Ver valor total del inventario:
1. En la pantalla de Inventario, abre el menú de opciones.
2. Selecciona "Valor del inventario".
3. Se mostrará un cuadro de diálogo con el valor total de todos los productos en Lempiras.
4. Haz clic en "Aceptar" para cerrar.

Exportar el inventario a PDF:
1. En Inventario, abre el menú de opciones.
2. Selecciona "Exportar inventario PDF".
3. Se abrirá una vista previa del documento con la información del inventario.
4. Desde esa pantalla podrás enviarlo a impresión o guardarlo, según las opciones disponibles en el sistema.

5.4 Registrar adquisiciones (compras)
1. Desde el menú principal, selecciona "Adquisiciones" o "Compras".
2. Elige el proveedor correspondiente.
3. Agrega los productos que se están comprando, indicando cantidades y precios de compra.
4. Revisa el resumen de la compra.
5. Haz clic en "Guardar" para registrar la compra y actualizar el inventario.

5.5 Registrar una venta

Preparación:
1. Asegúrate de que la caja esté abierta. Si no lo está, ve a la pantalla "Caja" y realiza el procedimiento de apertura según la política interna de la empresa.

Registrar la venta:
1. Desde el menú principal, selecciona "Ventas".
2. Si la caja está cerrada, se mostrará un mensaje correspondiente; debes abrirla antes de continuar.
3. Para agregar productos a la venta:
   - En escritorio: utiliza el buscador de productos para encontrarlos por nombre y haz clic para agregarlos al carrito.
   - En dispositivos móviles: utiliza el botón flotante para escanear códigos de barras o buscar por nombre.
4. Ajusta la cantidad de cada producto con los botones "+" y "-".
5. Si corresponde, aplica descuentos a productos individuales mediante el botón de descuento (icono de etiqueta).
6. Verifica en el resumen:
   - Total de productos.
   - Subtotal.
   - Descuento total.
   - Total a pagar.
7. Haz clic en "Finalizar venta".

Completar los datos de pago:
1. En el formulario de finalizar venta:
   - Selecciona el método de pago (efectivo, tarjeta, transferencia).
   - Verifica el campo "Total".
   - (Opcional) Ingresa el nombre del cliente.
   - (Opcional u obligatorio según el monto) Ingresa el RTN del cliente.
   - Ingresa el monto recibido del cliente.
2. El sistema calculará automáticamente el cambio.
3. Revisa toda la información.
4. Haz clic en "Confirmar" para registrar la venta.
5. La aplicación mostrará un mensaje indicando que la venta se registró con éxito.

Impresión de factura (si aplica):
1. Después de guardar la venta, la aplicación puede preguntar si deseas generar la factura.
2. Si aceptas, se mostrará una vista de impresión con el detalle de la venta, lista para imprimir en impresora térmica o en PDF.

5.6 Registrar una devolución
1. Desde el menú principal, selecciona "Devoluciones".
2. Haz clic en "Nueva devolución".
3. Selecciona la venta original de la cual se hará la devolución (por número de factura o mediante búsqueda).
4. Marca los productos y las cantidades a devolver.
5. Revisa el resumen de la devolución.
6. Confirma la operación para que el sistema:
   - Registre la devolución.
   - Actualice el inventario.

5.7 Consultar historial de ventas
1. En el menú principal, selecciona "Historial de ventas".
2. Recorre la lista para ver ventas pasadas.
3. Aplica filtros por fecha si están disponibles.
4. Haz clic en alguna venta para ver su detalle o reimprimir, según las opciones de la pantalla.

5.8 Gestionar usuarios
1. Desde el menú principal, selecciona "Usuarios".
2. Para crear un nuevo usuario:
   - Haz clic en el botón de nuevo usuario.
   - Completa los datos (nombre, correo, contraseña, tipo de usuario y permisos).
   - Guarda los cambios.
3. Para editar un usuario:
   - Selecciónalo de la lista.
   - Modifica la información necesaria.
   - Guarda los cambios.

5.9 Configuración SAR
1. Desde el menú principal, selecciona "Configuración SAR".
2. Revisa la configuración actual (CAI, rango de facturas, fecha límite, etc.).
3. Si necesitas actualizar algún dato, edítalo con cuidado según los documentos de autorización.
4. Haz clic en guardar.
5. Asegúrate de que los rangos de facturación coincidan con los autorizados oficialmente.

7. MENSAJES DEL SISTEMA

La aplicación muestra mensajes informativos, de advertencia y de error para guiar al usuario. Algunos ejemplos habituales:

7.1 Mensajes de éxito
- "Usuario administrador creado exitosamente. Inicie sesión."
  - Indica que se completó la configuración inicial del usuario administrador.
- "Venta registrada con éxito."
  - La venta se guardó correctamente.
- "Producto agregado" o mensajes similares:
  - Confirman que una acción se realizó satisfactoriamente.

7.2 Mensajes de advertencia
- "Credenciales incorrectas."
  - El correo o la contraseña no son válidos.
- "El usuario no se encuentra activo."
  - El usuario existe, pero está deshabilitado; debe contactar al administrador.
- "No tienes permiso para acceder a Inventario / Ventas / etc."
  - Indica que el usuario actual no tiene permisos para ese módulo.
- "Producto con código X no encontrado."
  - No se encontró el producto buscado en inventario.
- "Debe seleccionar al menos un producto."
  - Se intenta finalizar una venta sin productos seleccionados.
- "El RTN es obligatorio para montos mayores a L. XXXXX."
  - Es necesario ingresar el RTN del cliente para ventas de importe alto, según normativa.

7.3 Mensajes de error
- "Error al iniciar sesión."
  - Ocurrió un problema general al validar credenciales.
- "No hay configuración SAR activa."
  - Falta la configuración fiscal necesaria para generar facturas; debe configurarse en "Configuración SAR".
- "No se pudo generar el número de factura. Verifique rangos SAR."
  - El rango de facturas se agotó o está mal configurado.
- "Error al registrar la venta."
  - Fallo en el proceso de guardado de la venta; se recomienda contactar al área de sistemas.
- "Stock insuficiente para el producto X."
  - No hay suficiente inventario para completar la venta o acción.

7.4 Cómo interpretar los tipos de mensajes
- Mensajes de éxito: indican que todo salió bien; generalmente tienen un estilo visual en color verde.
- Mensajes de advertencia: indican una situación que requiere atención, pero que se puede corregir (color amarillo/naranja).
- Mensajes de error: indican un problema que impide completar la acción (color rojo).

8. SOLUCIÓN DE PROBLEMAS

A continuación se presentan algunos problemas frecuentes y cómo resolverlos:

Problema 1: No puedo iniciar sesión.
Posibles causas:
- El correo o la contraseña son incorrectos.
- El usuario está inactivo.
Solución:
1. Verifica que estés escribiendo correctamente el correo (sin espacios y con el dominio correcto).
2. Verifica que la contraseña esté bien escrita; utiliza el icono de "ojo" para verla si es necesario.
3. Si el problema persiste, usa la opción "¿Olvidaste tu contraseña?" para restablecerla, o contacta al administrador para verificar tu estado de usuario.

Problema 2: Aparece el mensaje "No tienes permiso para acceder a..." al entrar a un módulo.
Posibles causas:
- Tu usuario no tiene asignado el permiso necesario.
Solución:
1. Comunícate con el administrador del sistema para que revise y actualice tus permisos en la pantalla de "Usuarios".

Problema 3: No puedo registrar una venta porque dice que la caja está cerrada.
Posibles causas:
- No se ha abierto la caja del día.
Solución:
1. Ve a la pantalla "Caja".
2. Realiza el procedimiento de apertura de caja (según las políticas internas).
3. Regresa a la pantalla de "Ventas" y vuelve a intentar.

Problema 4: El sistema indica "No hay configuración SAR activa" o "No se pudo generar el número de factura".
Posibles causas:
- No se ha configurado el SAR.
- El rango de facturas autorizado se agotó.
Solución:
1. Inicia sesión con un usuario administrador.
2. Entra a "Configuración SAR".
3. Revisa y actualiza la información (CAI, rangos, fechas) según la documentación oficial otorgada por la autoridad fiscal.
4. Guarda los cambios y vuelve a intentar registrar la venta.

Problema 5: Me aparece "Stock insuficiente" al intentar vender un producto.
Posibles causas:
- La cantidad solicitada supera el stock disponible.
Solución:
1. Revisa la cantidad de productos en el carrito.
2. Reduce la cantidad o selecciona otro producto alternativo.
3. Si el stock es realmente bajo, revisa en Inventario si se requiere una nueva adquisición/compra.

Problema 6: No veo mis productos en el inventario.
Posibles causas:
- Los filtros de búsqueda están aplicados.
- El inventario está vacío.
Solución:
1. Borra el texto del campo de búsqueda para ver la lista completa.
2. Si el inventario sigue vacío, usa la opción "Agregar producto" para registrar nuevos productos.

Problema 7: La aplicación no abre en Windows.
Posibles causas:
- Falta de permisos.
- Bloqueo por antivirus.
- Archivo dañado.
Solución:
1. Asegúrate de ejecutar la aplicación con un usuario de Windows con permisos adecuados.
2. Revisa si el antivirus está bloqueando el programa; si es así, pide al área de sistemas que lo marque como seguro.
3. Si se sospecha de archivo dañado, solicita una nueva copia de la aplicación al área de sistemas.

9. PREGUNTAS FRECUENTES (FAQ)

Pregunta 1: ¿Puedo usar la aplicación en más de una computadora?
Respuesta:
Sí, es posible, siempre que el área de sistemas lo haya configurado así (por ejemplo, con base de datos compartida o copias sincronizadas). Consulta con tu administrador de sistemas antes de instalar la aplicación en otra máquina.

Pregunta 2: ¿Qué debo hacer cuando cambio de cajero?
Respuesta:
El cajero saliente debe cerrar sesión desde el menú principal (opción "Cerrar sesión"). Luego, el nuevo cajero debe iniciar sesión con su propio usuario y, si corresponde, abrir una nueva caja o continuar con la caja existente, según las políticas internas.

Pregunta 3: ¿Cómo sé si estoy usando el modo noche (oscuro)?
Respuesta:
En el menú lateral hay una opción llamada "Modo noche". Si el interruptor está activado, la aplicación se muestra con fondo oscuro. Si prefieres fondo claro, desactiva el interruptor.

Pregunta 4: ¿Puedo modificar una venta ya registrada?
Respuesta:
La forma recomendada de corregir una venta es a través del módulo de "Devoluciones", registrando la devolución correspondiente. Para cambios más avanzados, deberá consultarse con el administrador del sistema, ya que puede implicar ajustes contables y de facturación.

Pregunta 5: ¿Qué sucede si se agota el rango de facturas?
Respuesta:
El sistema mostrará mensajes de error al intentar generar nuevas facturas. En ese caso, el administrador debe actualizar la configuración en "Configuración SAR" con un nuevo rango autorizado por la autoridad fiscal.

Pregunta 6: ¿Cómo puedo ver un resumen de las ventas realizadas?
Respuesta:
Usa la pantalla de "Historial de ventas" para ver las operaciones pasadas y la pantalla de "Estadísticas" para ver un resumen gráfico del desempeño del negocio.

Pregunta 7: ¿Es posible reimprimir una factura?
Respuesta:
Si la funcionalidad está habilitada, podrás seleccionar la venta correspondiente en "Historial de ventas" y utilizar la opción de reimpresión o generación de factura. Si no ves esta opción, consulta con el administrador del sistema.

Pregunta 8: ¿Qué hago si olvido mi contraseña?
Respuesta:
En la pantalla de inicio de sesión, haz clic en "¿Olvidaste tu contraseña?" y sigue los pasos indicados. Si no puedes completarlos, solicita ayuda al administrador para que restablezca tu contraseña desde el módulo de "Usuarios".

Pregunta 9: ¿Quién puede cambiar la configuración SAR y de la empresa?
Respuesta:
Sólo los usuarios con permisos de administrador pueden acceder y modificar la configuración SAR y los datos de la empresa. Si necesitas un cambio, comunícalo al administrador del sistema.

Pregunta 10: ¿Qué pasa si cierro la aplicación mientras estoy trabajando?
Respuesta:
Al volver a abrirla, deberás iniciar sesión nuevamente. Las operaciones que ya se guardaron (ventas, inventario, etc.) se mantienen en la base de datos. Si estabas a mitad de una operación no guardada (por ejemplo, llenando un formulario), es posible que tengas que repetirla.
